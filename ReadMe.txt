NTB Stamp Duty And Registration Details.

- To run this project make sure to have xampp installed

- move project folder to the following directory:
	localdisk C:/xampp/htdocs/

- Run the xampp application

- Start Apache and MySQL

- open your browser, type in localhost/phpmyadmin

- create a database called ntb_stampduty

- import ntb_stampduty.sql file to the database created (ntb_stampduty)

- open your browser, type in localhost/(name of project root folder)
	in this case, type in localhost/NTB/
